SimpleMail

SimpleMail is (yet another) simple Java SMTP email sender.

Usage example

Create a SimpleMail instance that will send email using localhost:

Mailer mailer = new Mailer("localhost");

Construct an email:

Mail mail = new Mail()
            .from("m@test.com")
            .addTo("t@test.com")
            .withSubject("Hi")
            .withText("This is your email");

Send the email:

mailer.send(mail);
